// Classical Encryption
The Caesar cipher shifts every letter by a fixed amount — the key. Julius Caesar used a shift of 3. It's trivially breakable (only 25 possible keys), but contains the core idea of all modern encryption: transform plaintext into ciphertext using a key.

// SHA-256 Hash Visualizer
